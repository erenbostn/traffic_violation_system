# Contributing

Thanks for taking the time to contribute.

## Development setup

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Running

```bash
python src/main.py
```

## Pull requests

- Keep PRs small and focused.
- Include a clear description and screenshots/short clips when relevant.
- Avoid committing generated files in `outputs/` and large videos under `data/videos/`.

## Code style

- Prefer clear, descriptive names.
- Keep functions small; avoid unnecessary complexity.
